Temel konuları öğrenin: Merhaba Dünya, Kaçış Sekansları ve Değişken Yazdırma

👋Merhaba Dünya Programı

C programlamaya ilk adım! Her programcının yazdığı ilk program olan "Merhaba Dünya" örneği:

#include <stdio.h> int main() { printf("Merhaba Dünya!\n"); return 0; }
Merhaba Dünya!

🔍 Satır Satır Kod Açıklaması:

  • 1. satır: #include <stdio.h> - Standart giriş/çıkış kütüphanesini programa dahil eder (printf fonksiyonu için gerekli)
  • 3. satır: int main() - Programın başlangıç noktası olan ana fonksiyonu tanımlar
  • 4. satır: { - Ana fonksiyonun başlangıcını belirtir
  • 5. satır: printf("Merhaba Dünya!\n"); - Ekrana "Merhaba Dünya!" yazısını yazdırır ve yeni satıra geçer (\n)
  • 6. satır: return 0; - Programın başarıyla sonlandığını işletim sistemine bildirir
  • 7. satır: } - Ana fonksiyonun sonunu belirtir

🔤Kaçış Sekansları (Escape Sequences)

Kaçış sekansları, özel karakterleri ve formatları temsil etmek için kullanılır:

Kaçış Sekansı Açıklama Örnek Kullanım
\n Yeni satır printf("Satır 1\nSatır 2");
\t Tab (sekme) printf("Ad:\tAhmet");
\" Çift tırnak printf("\"Merhaba\"");
\\ Ters slash printf("C:\\Program Files");
\r Satır başı printf("Test\rYeni");
#include <stdio.h> int main() { printf("Birinci satır\n"); printf("İkinci\tsatır\n"); printf("\"Tırnak içinde metin\"\n"); printf("Dosya yolu: C:\\Users\\Kullanici\n"); return 0; }
Birinci satır İkinci satır "Tırnak içinde metin" Dosya yolu: C:\Users\Kullanici

🔍 Satır Satır Kod Açıklaması:

  • 1. satır: #include <stdio.h> - Standart giriş/çıkış kütüphanesini dahil eder
  • 3. satır: int main() - Ana fonksiyon tanımı
  • 4. satır: { - Fonksiyon bloğunun başlangıcı
  • 5. satır: printf("Birinci satır\n"); - "Birinci satır" yazdırır ve yeni satıra geçer
  • 6. satır: printf("İkinci\tsatır\n"); - "İkinci" yazdırır, tab boşluğu bırakır, "satır" yazdırır ve yeni satıra geçer
  • 7. satır: printf("\"Tırnak içinde metin\"\n"); - Çift tırnak işaretleri ile birlikte metni yazdırır
  • 8. satır: printf("Dosya yolu: C:\\Users\\Kullanici\n"); - Ters slash karakterlerini yazdırır
  • 9. satır: return 0; - Programın başarıyla sonlandığını belirtir
  • 10. satır: } - Fonksiyon bloğunun sonu

📊Değişken Verilerini Yazdırma

C'de farklı veri tiplerindeki değişkenleri yazdırmak için format belirleyiciler kullanılır:

Format Belirleyici Veri Tipi Açıklama
%d int Tam sayı
%f float Ondalıklı sayı
%c char Tek karakter
%s string Metin dizisi
%lf double Çift hassasiyetli ondalık
#include <stdio.h> int main() { // Değişken tanımlamaları int yas = 25; float boy = 1.75; char cinsiyet = 'E'; char isim[] = "Ahmet"; double maas = 5500.50; // Değişkenleri yazdırma printf("İsim: %s\n", isim); printf("Yaş: %d\n", yas); printf("Boy: %.2f metre\n", boy); printf("Cinsiyet: %c\n", cinsiyet); printf("Maaş: %.2lf TL\n", maas); return 0; }
İsim: Ahmet Yaş: 25 Boy: 1.75 metre Cinsiyet: E Maaş: 5500.50 TL

🔍 Satır Satır Kod Açıklaması:

  • 1. satır: #include <stdio.h> - Standart giriş/çıkış kütüphanesini dahil eder
  • 3. satır: int main() - Ana fonksiyon tanımı
  • 4. satır: { - Fonksiyon bloğunun başlangıcı
  • 5. satır: // Değişken tanımlamaları - Yorum satırı (derleyici tarafından göz ardı edilir)
  • 6. satır: int yas = 25; - Tam sayı tipinde "yas" değişkeni tanımlar ve 25 değerini atar
  • 7. satır: float boy = 1.75; - Ondalıklı sayı tipinde "boy" değişkeni tanımlar ve 1.75 değerini atar
  • 8. satır: char cinsiyet = 'E'; - Karakter tipinde "cinsiyet" değişkeni tanımlar ve 'E' karakterini atar
  • 9. satır: char isim[] = "Ahmet"; - Karakter dizisi (string) tipinde "isim" değişkeni tanımlar ve "Ahmet" metnini atar
  • 10. satır: double maas = 5500.50; - Çift hassasiyetli ondalık tipinde "maas" değişkeni tanımlar ve 5500.50 değerini atar
  • 12. satır: // Değişkenleri yazdırma - Yorum satırı
  • 13. satır: printf("İsim: %s\n", isim); - "İsim: " yazdırır, %s yerine isim değişkeninin değerini koyar ve yeni satıra geçer
  • 14. satır: printf("Yaş: %d\n", yas); - "Yaş: " yazdırır, %d yerine yas değişkeninin değerini koyar ve yeni satıra geçer
  • 15. satır: printf("Boy: %.2f metre\n", boy); - "Boy: " yazdırır, %.2f yerine boy değişkeninin değerini (2 ondalık basamakla) koyar ve yeni satıra geçer
  • 16. satır: printf("Cinsiyet: %c\n", cinsiyet); - "Cinsiyet: " yazdırır, %c yerine cinsiyet değişkeninin değerini koyar ve yeni satıra geçer
  • 17. satır: printf("Maaş: %.2lf TL\n", maas); - "Maaş: " yazdırır, %.2lf yerine maas değişkeninin değerini (2 ondalık basamakla) koyar ve yeni satıra geçer
  • 19. satır: return 0; - Programın başarıyla sonlandığını belirtir
  • 20. satır: } - Fonksiyon bloğunun sonu

💡 Önemli Notlar:

  • %.2f - Ondalık kısmı 2 basamakla sınırlar
  • %s - String (karakter dizisi) için kullanılır
  • %c - Tek karakter için kullanılır
  • %d - Tam sayılar için kullanılır
  • Format belirleyicilerin sırası, değişkenlerin sırasıyla eşleşmelidir
